**_Lami_** is a functional programming language based on lambda-calculus.

# How to run?

Clone the repo, and then run

```
cargo run
```

# Project structure

- `sl.lami`: Standard library
- `source.lami`: Source file that gets interpreted
- `src/`: Source code of the interpreter
